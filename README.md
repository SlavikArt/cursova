# ПВНЗ «ЄВРОПЕЙСЬКИЙ УНІВЕРСИТЕТ»
## КАФЕДРА КОМП’ЮТЕРНИХ НАУК ТА ПРОГРАМНОЇ ІНЖЕНЕРІЇ

**Курсова робота з дисципліни "Низькорівневе програмування"**

**Тема:** 20. Низькорівнева реалізація ефекту «дихаючого» світлодіода з використанням прямої маніпуляції портами, переривань та Watchdog-таймера

**Виконав:** Арцибашев В'ячеслав Сергійович  
**Група:** 242/1 он  
**Науковий керівник:** Яровий Р.О.

---

## Опис проекту
Система реалізує ефект плавної зміни яскравості світлодіодів (Sine, Triangle) з можливістю синхронізації.

**Розширений Функціонал:**
* Watchdog Timer
* Direct Port Manipulation
* Interrupts
* EEPROM
